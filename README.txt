Este código é um pequeno jogo de perguntas e respostas que usa a API 'ttps://the-trivia-api.com/api/questions' para fornecer as perguntas.
O jogo tem 10 perguntas aleatórias, cada uma com 4 opções de resposta, apenas uma é a resposta correta
As perguntas e opções são exibidas ao jogador e ele deve escolher a opção correta digitando o número correspondente
Após cada resposta, o jogo informa se a resposta está correta ou não. No final das 10 perguntas é exibido ao jogador quantas perguntas ele acertou
O código usa as bibliotecas "requests" e "json" para fazer a solicitação e processar a resposta da API e usa a biblioteca "random" para embaralhar as opções de resposta.
